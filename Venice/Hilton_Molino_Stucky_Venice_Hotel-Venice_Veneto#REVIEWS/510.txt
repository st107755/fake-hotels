I would never stay here again. The location is not the best, its far away from everything, so if you like to be isolated and in a quiet area, this should be good, but if you like to walk around and be where the main action is, this place is not for you. Being in Venice the shuttles can be an issue, so I prefer to be on the main land, taxis are crazy expensive!. The rooms are ok, depending on which you get, I had to change mine three times, some are tiny with tiny bathtubs and some are almost double in size with a shower and a bathtub (funny enough all for the same price which is not fair at all!). The rooms are clean and the bed is comfortable, the hotel is full of mosquitos, which is a horrible experience!
